Code Kata - 100 door/bulb problem.

October 13, 2015 • [programming] • [code] • [kata] • [problem-solving] • Anubhav Saini

As I was typing, door doesn’t sound like doar. Hmm weird English language.

Anyway, The problem is very simple as described here.

You are given a 100 (say) of doors or bulbs and you have to toggle them, iteratively. You will iterate through them 100 (usually) times and toggle their state. If closed/off, open/on them and vice versa.

This is basicall a mathematical problem that can be solved without getting into mathematics behind it.

In the end you will need to tell what is the state of the a given door or all of the doors or count of closed or open ones. (I dropped bulb from statement and we will concentrate on doors in this post moving on)

Official problem statement

Problem: You have 100 doors in a row that are all initially closed. You make 100 passes by the doors. The first time through, you visit every door and toggle the door (if the door is closed, you open it; if it is open, you close it). The second time you only visit every 2nd door (door #2, #4, #6, …). The third time, every 3rd door (door #3, #6, #9, …), -etc, until you only visit the 100th door.

Question: What state are the doors in after the last pass? Which are open, which are closed?

Solution statement

As I said, this is a mathematical problem, but we will refrain from it initially. One easy way to solve is as follows:

    for iteration = 1; iteration <= Doors.Count; iteration += 1
        for skip = iteration - 1, curr = skip; curr < Doors.Count; curr += skip
            Doors[curr] ^= 1

Where ^ indicates a XOR operation, so the whole line becomes a toggling function. What we do basically is that, for 100 iterations, we decide which door to start from and then based on the iteration number, we decide which doors to toggle.

I have made a mistake in logic above. What will happen is that above logic will sit in a forever loop.

Here’s the updated version:

    for iteration = 1; iteration <= Doors.Count; iteration += 1
        for skip = iteration, curr = skip-1; curr < Doors.Count; curr += skip
            Doors[curr] ^= 1

Let’s see the code in C# first.

C#

    using System;

    public class Test
    {
        public static void Main()
        {
            bool[] Doors = new bool[100];
            for(int iteration = 1; iteration <= Doors.Length; iteration += 1){
                for(int skip = iteration, currDoor = skip-1; currDoor < Doors.Length; currDoor += skip){
                    Doors[currDoor] ^= true;
                }
            }
            int i = 0;
            foreach(var door in Doors){
                i++;
                Console.WriteLine("{0} - {1}", door ? "Open" : "Closed", i);
            }
        }
    }
